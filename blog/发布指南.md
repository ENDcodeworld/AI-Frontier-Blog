## 知乎发布流程

### 步骤 1：打开知乎专栏
```
https://zhuanlan.zhihu.com/
```

### 步骤 2：创建文章
1. 点击右上角"写文章"
2. 选择对应专栏（或创建新专栏"AI 技术实战"）

### 步骤 3：粘贴内容
1. 标题：使用优化后的标题
2. 正文：从 Markdown 文件复制内容
3. 格式调整：
   - 代码块使用 ```language 格式
   - 图片使用拖拽上传
   - 链接添加为超链接

### 步骤 4：添加标签
```
#人工智能 #开源项目 #技术分享 #AI #编程
```

### 步骤 5：发布设置
- 原创声明：✓ 勾选
- 允许转载：✓ 勾选（注明出处）
- 定时发布：可选（建议工作日 9-10 点）

### 步骤 6：发布后推广
- 分享到朋友圈
- 分享到相关知乎圈子
- 回复评论互动

## 掘金发布流程

### 步骤 1：打开掘金
```
https://juejin.cn/
```

### 步骤 2：创作文章
1. 点击右上角"创作中心"
2. 选择"写文章"

### 步骤 3：填写信息
- 标题：优化后的标题
- 分类：选择对应分类（后端/前端/AI）
- 标签：添加 3-5 个标签
- 封面：选择吸引人的封面图

### 步骤 4：内容编辑
1. 使用 Markdown 编辑器
2. 代码块自动高亮
3. 图片上传到掘金图床

### 步骤 5：发布
- 预览检查
- 点击发布

### 步骤 6：加入专题
- 投稿到"AI 技术"、"开源项目"等专题
- 增加曝光度

## GitHub Discussions 发布流程

### 步骤 1：打开项目
```
https://github.com/yourname/your-project/discussions
```

### 步骤 2：新建讨论
1. 点击"New discussion"
2. 选择分类：Show and tell / General

### 步骤 3：填写内容
```markdown
Title: [Project Name] - 项目介绍

Body:
## 项目简介
（从博客摘要）

## 核心功能
（功能列表）

## 快速开始
（安装和使用说明）

## 技术栈
（技术列表）

## 项目链接
- GitHub: [链接]
- 详细博客：[知乎/掘金链接]

## 欢迎贡献
（贡献指南）
```

### 步骤 4：添加标签
- `announcement`
- `project-showcase`
- `help-wanted`

### 步骤 5：发布并置顶
- 发布后设置为置顶讨论

## V2EX 发布流程

### 步骤 1：打开 V2EX
```
https://www.v2ex.com/
```

### 步骤 2：选择节点
- 创意工作者
- 程序员
- 分享创造

### 步骤 3：发帖
```
标题：[项目名称] - 一句话亮点

正文：
项目简介
核心功能
技术亮点
项目链接
截图展示
```

### 步骤 4：注意事项
- 避免过度营销
- 真诚分享技术细节
- 积极回复评论
- 遵守社区规则

## 常见问题

### Q1: 知乎审核不通过？
- 检查是否有敏感词
- 确保原创声明
- 避免过度引流

### Q2: 掘金推荐量低？
- 优化标题和封面
- 选择合适标签
- 投稿到热门专题

### Q3: GitHub 没人关注？
- 完善 README
- 添加 Demo 和截图
- 在相关社区分享

### Q4: V2EX 被删帖？
- 避免营销语气
- 真诚分享技术
- 遵守版规
