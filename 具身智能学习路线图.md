## 什么是具身智能

**具身智能 (Embodied AI)** = AI 大模型 + 机器人身体

让 AI 不仅能"思考"，还能通过物理身体与环境交互、完成任务。

### 核心能力
- 🧠 **感知**：视觉、听觉、触觉理解环境
- 🤔 **决策**：基于大模型规划任务
- 🦾 **执行**：控制机械臂、轮式底盘等执行动作
- 🔄 **学习**：通过试错/模仿学习优化行为

### 典型应用
- 家庭服务机器人（扫地、整理、陪伴）
- 工业机械臂（分拣、装配）
- 自动驾驶车辆
- 人形机器人（特斯拉 Optimus、Figure 01）

## 3 个月学习计划

### 📅 第 1 个月：打基础

#### 第 1-2 周：Python + 深度学习复习
```
✅ Python 进阶
   - NumPy 数组操作
   - Matplotlib 可视化
   - 面向对象编程

✅ 深度学习核心
   - 神经网络基础（前向/反向传播）
   - CNN（图像分类）
   - Transformer（注意力机制）

📚 推荐资源：
   - 李宏毅《机器学习》YouTube 课程
   - PyTorch 官方教程 (pytorch.org/tutorials)
   - 3Blue1Brown《神经网络详解》
```

#### 第 3-4 周：PyTorch 实战
```
✅ 动手项目：
   - 手写数字识别 (MNIST)
   - 图像分类 (CIFAR-10)
   - 简单目标检测

✅ 关键技能：
   - Dataset/DataLoader
   - 模型定义与训练循环
   - 损失函数与优化器
   - GPU 加速训练

🎯 产出：GitHub 仓库，包含 3 个完整项目代码
```

## 实战项目推荐

### 🟢 入门级（1-2 周）

1. **MNIST 手写识别** — 深度学习 Hello World
2. **ROS2 仿真小车** — 控制小车前后左右移动
3. **YOLO 目标检测** — 识别图片中的物体

### 🟡 进阶级（3-4 周）

1. **机械臂仿真抓取**
   - 工具：PyBullet + PyTorch
   - 任务：训练机械臂抓取指定颜色物体

2. **视觉 SLAM 建图**
   - 工具：ORB-SLAM3 + ROS2
   - 任务：用摄像头构建室内地图

3. **语言控制机器人**
   - 工具：LLM API + ROS2
   - 任务：说"去厨房"，机器人导航到厨房

### 🔴 高级级（6-8 周）

1. **端到端策略学习**
   - 工具：Isaac Sim + PPO
   - 任务：从图像直接输出动作，完成复杂操作

2. **多机器人协作**
   - 工具：ROS2 + 强化学习
   - 任务：多个机器人协作搬运物体

3. **人形机器人控制**
   - 工具：MuJoCo + 模仿学习
   - 任务：让人形机器人学会走路

## 📝 每日学习建议

```
⏰ 时间分配（每天 3 小时）
├── 1h 理论学习（视频/书籍）
├── 1.5h 动手实践（写代码/跑实验）
└── 0.5h 总结记录（写笔记/博客）

📅 每周节奏
├── 周一 ~ 周五：学习 + 实践
├── 周六：复盘 + 补漏
└── 周日：休息 / 看论文

💡 关键原则
├── 不要只看不练！每学一个概念就写代码验证
├── 遇到问题先 Google/StackOverflow，再问人
├── 定期整理笔记，形成知识体系
└── 多做项目，少刷课程（课程是手段，项目是目的）
```
